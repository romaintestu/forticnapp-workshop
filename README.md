# FortiCNAPP Workshop: AWS Integration

Hands-on labs for FortiCNAPP (Fortinet Cloud Native Application Protection Platform) cloud security integration with AWS.

## Overview

Step-by-step instructions for integrating FortiCNAPP with AWS for cloud security monitoring, compliance, and threat detection.

## Prerequisites

- AWS account with appropriate permissions
- FortiCNAPP account access
- Basic knowledge of AWS services (CloudTrail, Config, EC2)
- Command-line access (Linux/Mac/Windows with WSL)
